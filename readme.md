# Otimização de Fertilizantes e Saúde do Solo

## 🎯 Objetivo do Projeto
Este projeto visa resolver uma dor latente no agronegócio: o uso ineficiente de insumos químicos. Através de análise de dados de solo, o sistema recomenda a dosagem ideal de fertilizantes, reduzindo custos e impacto ambiental.

## 🛠️ Tecnologias Utilizadas
- **Python**: Lógica central e processamento.
- **JSON**: Armazenamento de parâmetros técnicos de referência.
- **Oracle Database**: Persistência de dados históricos de análises.

## 📂 Estrutura de Pastas
- `src/`: Código-fonte do sistema.
- `src/data/`: Arquivos de configuração e parâmetros (JSON).
- `src/database.py`: Subalgoritmos de conexão com banco de dados.
- `src/processamento.py`: Lógica de cálculos e tratamento de estruturas.
